Procedure to migrate from JAHIA_EE_V6.1.1_3_r37857 to JAHIA_EE_V6.1.1_4_r38450 :
  1. Remove the following files from webapps\ROOT\WEB-INF\lib\
    1. jahia-impl-*.jar
    2. jahia-taglib-*.jar
    3. jahia-gwt*.jar
    4. jahia-api*.jar

  2. and from webapps\ROOT\WEB-INF\var\shared_templates\
    1. web-templates-*.war
    2. test-templates*.war
    3. web-templates-extension-*.war
    4. blog_templates-*.war

  3. and
    1. Remove the following folders :
      • templates/web-templates
      • templates/web-templates-extension

    2. remove the content of the <JAHIA_INSTALLATION>/work/ and <JAHIA_INSTALLATION>/temp/ folders

  4. Manually modify the following configuration file:
    • Jahia_properties_3.txt for WEB-INF\etc\config\jahia.properties
    • Applicationcontext_indexationpolicy_1.txt for WEB-INF\etc\spring\applicationcontext-indexationpolicy.xml
    • Applicationcontext_services_3.txt for WEB-INF\etc\spring\applicationcontext_services.xml

  5. Copy the content of the output folder into your webapps\ROOT\ folder

    6. !!We may have changed our internal ressources bundles, if you customized them please check the output\WEB-INF\classes.

    !!If you have modified one the files present in the folder you need to remove it from the output folder and manually report the changes

    !!We also changed default config of FCK Editor, if you customized it please check the output\htmleditors\fckeditor

  6. Launch Jahia
