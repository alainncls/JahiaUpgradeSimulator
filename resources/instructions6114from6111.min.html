Procedure to migrate from JAHIA_EE_V6.1.1_1_r35016 to JAHIA_EE_V6.1.1_4_r38450 :
  1. Remove the following files from webapps\ROOT\WEB-INF\lib\
    1. jahia-impl-*.jar
    2. jahia-taglib-*.jar
    3. jahia-gwt*.jar
    4. jahia-api*.jar

  2. and from webapps\ROOT\WEB-INF\var\shared_templates\
    1. web-templates-*.war
    2. test-templates*.war
    3. web-templates-extension-*.war
    4. blog_templates-*.war

  3. and
    1. remove html/startup/images/flight.gif
    2. Remove the following folders :
      • templates/web-templates
      • templates/web-templates-extension
      • templates/web-templates-extension
      • templates/blog_templates

    3. Remove the following files :
      • WEB-INF/var/patches/sql/mysql/jahia_urlmapping_creation.sql
      • WEB-INF/var/patches/sql/oracle/jahia_urlmapping_creation.sql
      • WEB-INF/var/patches/sql/hypersonic/jahia_urlmapping_creation.sql
      • WEB-INF/var/patches/sql/derby/jahia_urlmapping_creation.sql
      • WEB-INF/var/patches/sql/mssql/jahia_urlmapping_creation.sql
      • WEB-INF/var/patches/sql/postgresql/jahia_urlmapping_creation.sql
      • WEB-INF\classes\logger.properties

    4. remove the content of the <JAHIA_INSTALLATION>/work/ and <JAHIA_INSTALLATION>/temp/ folders
    5. remove the following file from your server lib (e.g. for tomcat <TOMCAT_HOME>/lib): pluto-container-2.0.0-RI.jar

  4. Manually modify the following configuration file:
    • Applicationcontext-services_1.txt for WEB-INF\etc\spring\applicationcontext_services.xml
    • Applicationcontext-services_2.txt for WEB-INF\etc\spring\applicationcontext_services.xml
    • Applicationcontext_services_3.txt for WEB-INF\etc\spring\applicationcontext_services.xml
    • Jahia_properties_1.txt for WEB-INF\etc\config\jahia.properties
    • Jahia_properties_2.txt for WEB-INF\etc\config\jahia.properties
    • Jahia_properties_3.txt for WEB-INF\etc\config\jahia.properties
    • Applicationcontext_indexationpolicy.txt for WEB-INF\etc\spring\applicationcontext-indexationpolicy.xml
    • Applicationcontext_indexationpolicy_1.txt for WEB-INF\etc\spring\applicationcontext-indexationpolicy.xml
    • Applicationcontext_basejahiaconfig.txt for WEB-INF\etc\spring\applicationcontext-basejahiaconfig.xml
    • web_xml for WEB-INF\web.xml

  5. Copy the content of the output folder into your webapps\ROOT\ folder

    6. !!We may have changed our internal ressources bundles, if you customized them please check the output\WEB-INF\classes.

    !!If you have modified one the files present in the folder you need to remove it from the output folder and manually report the changes

    !!We also changed default config of FCK Editor, if you customized it please check the output\htmleditors\fckeditor

  6. Launch Jahia

  7. Go to the patch management section and launch the scripts filereferences_35106.groovy and filereferences2_36783.groovy
