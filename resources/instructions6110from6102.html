Procedure to migrate from JAHIA_EE_V6.1.0_2_r32497 to JAHIA_EE_V6.1.1_r34653 :
  1. Remove the following jars from webapps\ROOT\WEB-INF\lib\
    • jahia-impl-*.jar
    • jahia-taglib-*.jar
    • jahia-gwt*.jar
    • jahia-api*.jar
    • drools-*.jar (4 jars)
    • jackrabbit-core-1.5.0.jar
    • mvel14-1.2.10.jar
    • jtidy-4aug2000r7-dev.jar
    • ehcache-1.5.0.jar
    • jsr107cache-1.0.jar
    • backport-util-concurrent-3.1.jar

  2. and from webapps\ROOT\WEB-INF\var\shared_templates\
    • web-templates-extension-*.war
    • web-templates-*.war
    • blog_templates-*.war

  3. Remove the following folders :
    • templates/web-templates-extension
    • templates/web_templates
    • templates/blog_templates

  4. Remove the content of the following folders :
    • <JAHIA_INSTALLATION>/Work folder
    • <JAHIA_INSTALLATION>/temp folder

    !! Due to some backend changes in how Jahia handles the cache structure, you need to remove completely the content of this folder otherwise Jahia will not be able to correctly cache the content. Make sure you do not miss this step !

    An issue impacting the correct behaviour of the cache="false" and cache="off" attributes in the templates has been fixed.We strongly recommend you check in your templates if these attributes are used. Make sure you actually intend to have these parts NOT cached by Jahia, otherwise you should remove them.

  5. Modify the following configuration file:
    • applicationcontext-services2.txt for WEB-INF\etc\spring\applicationcontext_services.xml
    • ehcache.txt for \WEB-INF\classes\ehcache-jahia.xml AND \WEB-INF\classes\ehcache-jahia_cluster.xml
    • jahia_properties_4.txt for WEB-INF\etc\config\Jahia.properties
    • jahia_properties_3.txt for WEB-INF\etc\config\Jahia.properties
    • workspace_xml.txt for WEB-INF\var\repository\workspaces\default\workspace.xml
    • applicationcontext-manager.txt for WEB-INF\etc\spring\applicationcontext_manager.xml

  6. Copy the content of the output folder into your webapps\ROOT\ folder

  7. Launch Jahia
