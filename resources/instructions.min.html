How to Install the Hotfix

Please read carefully the instructions before proceed.

Procedure to migrate :

  • 1. Remove the following jars from webapps\\ROOT\\WEB-INF\\lib\\
    • a. jahia-gwt-*.jar
    • b. jahia-impl-*.jar
    • c. jahia-taglib-*.jar
  • 2. Copy the content of the output folder into your webapps\\ROOT\\ folder
  • 3. Launch Jahia
