Procedure to migrate from Jahia_EE_V6_SP1_r28889 to jahia-hotfix-6.0.1_1_r29690:

1. Remove the following jars from webapps\ROOT\WEB-INF\lib\
  • jahia-impl-*.jar
  • jahia-taglib-*.jar
and from webapps\ROOT\WEB-INF\var\shared_templates
  • web-templates-6.0-*.jar
  • test-templates-6.0*.jar
2. Modify the following configuration file:
  • UnixDiff_jahia_properties.txt for WEB-INF\etc\config\jahia.properties
  • UnixDiff_Applicationcontext_administration.txt for WEB-INF\etc\spring\applicationcontext-administration.xml
    • For Linux environment :
  • UnixDiff_catalina_sh.txt for <JAHIA INSTALLATION>\bin\catalina.sh
    • For Windows environment :
  • UnixDiff_catalina_bat.txt for <JAHIA INSTALLATION>\bin\catalina.bat
    3. Copy the content of the output folder into your webapps\ROOT\ folder
    4. Launch Jahia
