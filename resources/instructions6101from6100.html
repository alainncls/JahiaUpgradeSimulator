Procedure to migrate from Jahia_EE_V6.1.0_r31127 to JAHIA_EE_V6.1.0_1_r31787 :
  1. Remove the following jars from webapps\ROOT\WEB-INF\lib\
    1. jahia-impl-*.jar
    2. jahia-taglib-*.jar
    3. jahia-gwt*.jar

  2. and from webapps\ROOT\WEB-INF\var\shared_templates\
    1. web-templates-extension-*.war
    2. blog_templates-*.war
    3. Remove the following folder : templates/web-templates-extension and templates/blog_templates
    4. remove the content of the <JAHIA_INSTALLATION>/Work folder

  3. Modify the following configuration file:
    • UnixDiff_Applicationcontext-services_xml.txt for WEB-INF\etc\spring\applicationcontext_services.xml
    • UnixDiff_Jahia_properties.txt for WEB-INF\etc\config\Jahia.properties

  4. Copy the content of the output folder into your webapps\ROOT\ folder

  5. To benefit of some indexes optimisation, you need to run the sql script corresponding to your database in WEB-INF\var\patches\sql\<DATABASE>\jahia-schema-index_31435.sql

  6. Launch Jahia
