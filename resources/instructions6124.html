Procedure to migrate from JAHIA_EE_V6.1.2_3_r43818 to JAHIA_EE_V6.1.2_4_r46320:
  1. Remove the following jars from webapps/ROOT/WEB-INF/lib/
    1. jahia-impl-*.jar
    2. jahia-taglib-*.jar
    3. jahia-gwt*.jar
    4. jahia-api*.jar
    5. jericho-html-*.jar
    6. ehcache-core-*.jar
    7. jackrabbit-core-ext-*.jar
    8. commons-dbcp-*.jar
    9. commons-pool-*.jar

  2. Remove the content of the following folder:
    • <JAHIA_INSTALLATION</Work
    • <JAHIA_INSTALLATION</temp

  3. Manually modify the following configuration file:
    • Jahia_properties3.txt for WEB-INF/etc/config/jahia.properties
    • Applicationcontext-services1.txt for WEB-INF/etc/spring/applicationcontext_services.xml
    • ehcache-jahia.txt for WEB-INF/classes/ehcache-jahia.xml
    • ehcache-jahia_cluster.txt for WEB-INF/classes/ehcache-jahia_cluster.xml

  4. Copy the content of the output folder into your webapps/ROOT/ folder

    5. We may have changed our internal ressources bundles, if you customized them please check the output/WEB-INF/classes. If you have modified one of the files present in the folder you need to remove it from the output folder and manually report the changes.

  5. To benefit our recent database improvement, you need to run the sql scripts corresponding to your database in:
    • WEB-INF/var/patches/sql/<DATABASE>/jahia_workflow_links_cleanup_43763.sql

  6. Launch Jahia
