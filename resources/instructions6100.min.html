HOW TO UPGRADE TO JAHIA_EE_V6 TO JAHIA_EE_V6_SP1

Please read carefully the instructions before proceed. (a txt version of the readme is also available in the \"jahia-hotfix-6.0.1_x\" root folder)

Procedure to migrate from jahia-hotfix-6.0.0_4_r28200 to Jahia_EE_V6_SP1_r28889:

1. Remove the following jars from webapps\\ROOT\\WEB-INF\\lib\\ :
  • jahia-gwt-*.jar
  • jahia-impl-*.jar
  • jahia-taglib-*.jar
  • jahia-api-*.jar
  • jackrabbit-core-ext-*.jar
  • jackrabbit-server-ext-*.jar
and from webapps\\ROOT\\WEB-INF\\var\\shared_templates :
  • web-templates-6.0-*.jar
  • test-templates-6.0*.jar
  • core-templates-6.0*.jar
2. Modify the following configuration file:
  • UnixDiff_Applicationcontext-services3_xml.txt for WEB-INF\\etc\\spring\\applicationcontext-services.xml
  • UnixDiff_web2_xml.txt for WEB-INF\\web.xml
    3. Copy the content of the output folder into your webapps\\ROOT\\ folder
    4. Launch Jahia
