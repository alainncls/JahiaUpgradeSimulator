How to Install the Hotfix

Please read carefully the instructions before proceed.

Procedure to migrate :

  • 1. Remove the following jars from webapps\\ROOT\\WEB-INF\\lib\\
    • a. jahia-gwt-*.jar
    • b. jahia-impl-*.jar
    • c. jahia-taglib-*.jar
    • d. jackrabbit-core-ext-*.
  • 2. Modify the following configuration file:
    • UnixDiff_web_xml.txt for WEB-INF\\web.xml
    • UnixDiff_applicationcontext-hibernate_xml.txt for WEB-INF\\etc\\spring\\applicationcontext-hibernate.xml
    • UnixDiff_Applicationcontext-services_xml.txt AND UnixDiff_Applicationcontext-services2_xml.txt for WEB-INF\\etc\\spring\\applicationcontext-services.xml
    • UnixDiff_applicationcontext-administration_xml.txt for WEB-INF\\etc\\spring\\applicationcontext-administration.xml
    • UnixDiff_applicationcontext-notification_xml.txt for WEB-INF\\etc\\spring\\applicationcontext-notification.xml
    • UnixDiff_ehcache-jahia_cluster_xml.txt for WEB-INF\\etc\\classes\\ehcache-jahia_cluster.xml
    • UnixDiff_JahiaInternalResources_properties.txt AND * UnixDiff_JahiaInternalResources_properties2.txt for WEB-INF\\classes\\JahiaInternalResources.properties
    • UnixDiff_JahiaTypesResources_properties.txt for WEB-INF\\classes\\JahiaTypesResources.properties
    • UnixDiff_common_properties.txt for WEB-INF\\classes\\jahiatemplates\\common.properties
  • 3. Copy the content of the output folder into your webapps\\ROOT\\ folder
  • 4. Launch Jahia
