Procedure to migrate from Jahia_EE_V6.1.1_r34653 to JAHIA_EE_V6.1.1_1_r35016 :
  1. Remove the following jars from webapps\ROOT\WEB-INF\lib\
    • jahia-impl-*.jar
    • jahia-taglib-*.jar
    • jahia-gwt*.jar
    • jahia-api*.jar

  2. and from webapps\ROOT\WEB-INF\var\shared_templates\
    • web-templates-*.war
    • test-templates*.war
    • Remove the following folders : templates/web-templates AND templates/test_templates
    • remove the content of the <JAHIA_INSTALLATION>/Work folder

  3. Manually modify the following configuration file:
    • Applicationcontext-services_xml.txt for WEB-INF\etc\spring\applicationcontext_services.xml
    • Jahia_properties.txt for WEB-INF\etc\config\Jahia.properties

  4. Copy the content of the output folder into your webapps\ROOT\ folder

  5. Launch Jahia
