Procedure to migrate from JAHIA_EE_V6.1.h1_2_r36264 to JAHIA_EE_V6.1.1_3_r37857 :
  1. Remove the following jars from webapps\ROOT\WEB-INF\lib\
    1. jahia-impl-*.jar
    2. jahia-taglib-*.jar
    3. jahia-gwt*.jar
    4. jahia-api*.jar

  2. and from webapps\ROOT\WEB-INF\var\shared_templates\
    1. web-templates-*.war
    2. test-templates*.war
    3. web-templates-extension-*.war
    4. blog_templates-*.war

  3. Remove the following files :
    • WEB-INF\classes\logger.properties

  4. Remove the following folders :
    • templates/web-templates
    • templates/test_templates
    • templates/web-templates-extension
    • templates/blog_templates

  5. remove the content of the <JAHIA_INSTALLATION>/work/ and <JAHIA_INSTALLATION>/temp/ folders

  6. Manually modify the following configuration file:
    • Jahia_properties_2.txt for WEB-INF\etc\config\jahia.properties
    • Applicationcontext_indexationpolicy.txt for WEB-INF\etc\spring\applicationcontext-indexationpolicy.xml
    • Applicationcontext_services_2.txt for WEB-INF\etc\spring\applicationcontext_services.xml
    • Applicationcontext_basejahiaconfig.txt for WEB-INF\etc\spring\applicationcontext-basejahiaconfig.xml
    • web_xml.txt for WEB-INF\web.xml

  7. Copy the content of the output folder into your webapps\ROOT\ folder

    8. !!We may have changed our internal ressources bundles, if you customized them please check the output\WEB-INF\classes.

    If you have modified one the files present in the folder you need to remove it from the output folder and manually report the changes!!

  8. Launch Jahia

  9. Go to the patch management section and launch the filereferences2_36783.groovy script
